# Core Traders - Industrial Supplier Platform

Professional React-based e-commerce platform for Core Traders industrial products.

## Features

- Product Catalog with Filtering
- Shopping Cart Management
- User Authentication
- Client Portal
- Responsive Design
- Context API State Management

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Project Structure

- `/src/components` - Reusable UI components
- `/src/pages` - Page components
- `/src/contexts` - Global state management
- `/src/services` - API calls
- `/src/styles` - Global styles
